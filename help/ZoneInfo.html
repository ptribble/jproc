ZoneInfo

ZoneInfo is a tabular display of running processes and some of their properties.

The fields displayed are:

  • ZONE: zone name
  • NPROC: number of processes belonging to this zone
  • nlwp: number of lwps (threads) belonging to this zone
  • SZ: total size of the processes belonging to this zone
  • RSS: total resident set size of the processes belonging to this zone
  • TIME: cpu time accumulated by the processes belonging to this zone

The rows can be sorted by selecting the column header.

The toolbar

The toolbar contains a small number of menus.

  • File - allows you to exit ZoneInfo.
  • Update every - allows you to change how rapidly the statistics are updated.
  • Help - shows this help file, and the CDDL license that this software is licensed under.
  • Users - allows you to select which users' processes are included.
