UserInfo

UserInfo is a tabular display of running processes and some of their properties.

The fields displayed are:

  • USER: user name
  • NPROC: number of processes belonging to this user
  • nlwp: number of lwps (threads) belonging to this user
  • SZ: total size of the processes belonging to this user
  • RSS: total resident set size of the processes belonging to this user
  • TIME: cpu time accumulated by the processes belonging to this user

The rows can be sorted by selecting the column header.

The toolbar

The toolbar contains a small number of menus.

  • File - allows you to exit UserInfo.
  • Update every - allows you to change how rapidly the statistics are updated.
  • Help - shows this help file, and the CDDL license that this software is licensed under.
  • Zones (only if multiple zones are running when UserInfo starts) - allows you to select which zones' processes are shown.
